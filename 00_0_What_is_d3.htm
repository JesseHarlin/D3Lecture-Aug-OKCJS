D3 [Data Driven Documents]


Jesse Harlin - OKC.js


D3 aims to solve a few problems

  • Provide a simple selector system for dealing with the DOM [like jQuery]
  • Ease the pain of working with SVG, by abstracting away the hard to read parts (like D)
  • Allow for dynamic properties with a functional data-bound callback system [resembles a lot of underscore]
  • Provide a 'data-first' and well defined pattern way of working with visualization [enter, update, transition, exit]
  • Add Zazzle™ to your page.
